# Alpha 版本

## Alpha 0.0.1更新日志

发布日期：2026-4-19

变化：
- 改为使用 Godot 4.6.2 进行开发（并且在不影响项目程序的情况下使用更新的版本）
- 新增了 1 种瓦片种类
- 新增了 6200 字剧情

## Alpha 0.0.2更新日志

发布日期：2026-5-1

变化：
- 使用 TileMap 搭建了测试地图
- 完成了对玩家移动和对事件触发点的测试
- 剧情大纲搭建完成，剩余所有剧情交由 Suflante 完成

计划更新：
- 使用双瓦片搭建地图（感谢 Godot Addon: [TileMapDual](https://github.com/pablogila/TileMapDual)）
